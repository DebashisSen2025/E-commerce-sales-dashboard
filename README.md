# E-Commerce Sales Dashboard

Interactive Power BI dashboard for analyzing e-commerce performance.

## Features

- Segment-based filtering (Consumer, Corporate, Home Office)
- Year-to-Date KPIs: Sales, Profit, Quantity, Margin
- Visual sales trend indicator with conditional formatting
- Top 5 & Bottom 5 products by YTD sales
- Sales distribution by category and region

## Dashboard Preview

![Dashboard screenshot](images/ecommerce-dashboard.png)

## Key Visuals

- **KPIs cards** with trend icons
- **Top/Bottom 5 products** – stacked bar charts
- **Category & Region breakdown** – donut charts

## Data Source

- Sales transactions (anonymized)
- Fields: Order Date, Segment, Category, Sub-Category, Product Name, Region, Sales, Profit, Quantity

## Requirements

- Power BI Desktop (May 2024 or newer recommended)

## How to Use

1. Open `E-Commerce Sales Dashboard.pbix`
2. Use the **Segment** slicer to filter data
3. Review YTD KPIs and trend indicators
4. Analyze top/bottom products and category/region distribution
